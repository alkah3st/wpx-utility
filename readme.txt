Utility functions for use in WPX Theme.

== Description ==

WP Extend (WPX) Utilities are helper functions for use with the WPX Theme.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `/wpx-utility/` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it!
